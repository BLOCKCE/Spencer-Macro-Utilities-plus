# Roblox Macro Client
An open-source Windows C++ Roblox ImGui Macro with many features.

To Compile, run the project in Visual Studio 2022 and build it.

### Is this A CHEAT???
No, it's a macro, it doesn't communicate with Roblox memory in any way.
